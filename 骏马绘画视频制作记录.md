## 💻 核心技术

### 1. SVG 绘制

```svg
<ellipse cx="400" cy="350" rx="120" ry="60" 
         fill="url(#bodyGrad)" stroke="#654321" stroke-width="3"/>
```

### 2. 分帧动画生成

```bash
# 生成 60 帧 SVG
for i in $(seq 1 60); do
    # 根据进度显示不同部分
    if [ $body_progress -eq 1 ]; then
        # 添加身体部分
    fi
done
```

### 3. SVG 转 PNG

```bash
rsvg-convert frame_001.svg -o frame_001.png -w 1920 -h 1080
```

### 4. FFmpeg 视频合成

```bash
ffmpeg -framerate 20 -i frame_%03d.png \
  -c:v libx264 -preset slow -crf 18 \
  -pix_fmt yuv420p output.mp4
```

## 💡 技术要点

### 1. SVG 分帧动画

通过控制每帧 SVG 中不同元素的显示/隐藏，实现逐步绘画效果。

### 2. 进度计算

```bash
progress=$((i * 100 / 60))  # 计算百分比
```

### 3. 条件渲染

```bash
body_progress=$((i > 5 ? 1 : 0))  # 第 5 帧后显示身体
```

### 4. FFmpeg 参数优化

- `-preset slow` - 更好的压缩率
- `-crf 18` - 高质量（18-23 范围）
- `-pix_fmt yuv420p` - 兼容性最佳
