- **现象**：使用 `--ref-audio` 参数克隆特朗普声音，生成的 wav 文件只有 70 字节
- **可能原因**：
  1. Noiz API 的声音克隆功能需要更高权限的 API Key
  2. 参考音频格式不兼容（mp3 vs wav）
  3. 参考音频太长（15MB 的演讲录音）
- **临时方案**：先用默认中文女声（af_sarah）发语音
- **后续**：需要查 Noiz 文档或测试更短的参考音频
