## 🇨🇳 中国特色用例详解（13 个）

### 1. 飞书 AI 助手 ⭐⭐
- **功能**：部署为飞书机器人，对话触发 AI 任务，支持文档自动化
- **适配点**：飞书开放平台集成

### 2. 钉钉 AI 助手 ⭐⭐
- **功能**：部署为钉钉机器人，Stream 模式无需公网 IP
- **适配点**：钉钉开放平台 + 内网穿透

### 3. 企业微信 AI 助手 ⭐⭐
- **功能**：企业微信中使用 AI，微信插件支持个人微信
- **适配点**：企业微信 API+ 微信插件

### 4. 早间简报（适配）⭐
- **功能**：每日定时推送到飞书/钉钉
- **适配点**：中文新闻源 + cron 配置

### 5. 办公自动化套件 ⭐⭐
- **功能**：邮件管理、文件整理、会议纪要、周报生成
- **适配点**：163/QQ/Outlook 邮箱支持

### 6. 会议纪要与待办自动化（适配）⭐⭐
- **功能**：会议转录自动生成纪要并创建任务
- **适配点**：飞书妙记/腾讯会议/钉钉集成

### 7. 多渠道 AI 客户服务（适配）⭐⭐⭐
- **功能**：企业微信/抖音/小红书多渠道客服自动化
- **适配点**：国内社交平台 API

### 8. 小红书内容自动化 ⭐
- **功能**：选题→文案→封面图→定时发布全流程
- **适配点**：小红书 MCP 集成

### 9. 播客制作流水线（适配）⭐⭐
- **功能**：选题到发布全流程播客制作
- **适配点**：小宇宙/喜马拉雅/B 站

### 10. AI 财报追踪器（适配）⭐⭐
- **功能**：A 股财报追踪，业绩预告/快报自动化
- **适配点**：AKShare 免费数据源

### 11. 开发前创意验证器（适配）⭐⭐
- **功能**：编码前自动扫描竞品
- **适配点**：百度指数/微信指数/V2EX/ 少数派

### 12. 多智能体协作操作系统 ⭐⭐⭐
- **功能**：专业分工、协同、稳定迭代的智能体系统
- **适配点**：国内生态完整适配

## 💡 快速开始建议

### 第一步：选择平台
- **国际用户**：Telegram / Discord / iMessage
- **国内用户**：飞书 / 钉钉 / 企业微信

### 第二步：安装基础技能
```bash
# 基础技能包
clawhub install agent-browser      # 浏览器自动化
clawhub install imap-smtp-email    # 邮件收发
clawhub install github             # GitHub 集成
```

### 第三步：选择一个用例开始
**推荐新手从⭐难度开始：**
1. 晨间简报（Custom Morning Brief）
2. 邮件摘要（Inbox De-clutter）

掌握基础后尝试⭐⭐和⭐⭐⭐用例，构建自己的自动化工作流。
