## ⏰ 教学流程

### 0-15 分钟：作业回顾

**活动：**
1. 播放 2 个优秀《静夜思》朗诵视频
2. 点评声音和肢体表现
3. 学生分享练习心得

**点评要点：**
```
✅ 语调有起伏
✅ 停顿恰当
✅ 手势配合诗句
✅ 眼神有交流
```

---

### 15-35 分钟：故事结构讲解

**Story Arc 故事弧光：**
```
        高潮
         /\
        /  \
       /    \
  开始      结局
   ↓          ↓
  介绍      解决
   ↓
  冲突
```

**四要素详解：**

| 要素 | 功能 | 示例 |
|------|------|------|
| **开始** | 介绍背景、人物 | "Last year, I met a person..." |
| **冲突** | 制造紧张、问题 | "But then something unexpected happened..." |
| **高潮** | 情感最高点 | "At that moment, I realized..." |
| **结局** | 解决、感悟 | "Now I understand that..." |

**细节描写技巧 - 五觉法：**

| 感官 | 问题 | 示例 |
|------|------|------|
| **视觉** | 你看到了什么？ | "The red lanterns were glowing..." |
| **听觉** | 你听到了什么？ | "I heard the sound of firecrackers..." |
| **嗅觉** | 你闻到了什么？ | "The smell of dumplings filled the air..." |
| **味觉** | 你尝到了什么？ | "The sweet taste of mooncake..." |
| **触觉** | 你感受到了什么？ | "The warm handshake made me feel..." |

---

### 35-50 分钟：案例分析

**观看视频："嫦娥探月"故事（3 分钟）**

**小组讨论（4 人一组）：**
```
1. 故事的开始是什么？
2. 冲突/挑战是什么？
3. 高潮在哪里？
4. 结局如何？
5. 哪个细节最打动你？
```

**分享环节：**
- 每组选代表分享分析结果
- 教师补充点评

**教师分析要点：**
```
开始：中国千年的飞天梦想
冲突：技术封锁、资金不足
高潮：嫦娥五号成功登月
结局：带回月壤，新的起点

细节亮点：
- "工程师们连续工作了 72 小时"
- "控制室里的掌声持续了 5 分钟"
```

---

### 50-80 分钟：5 分钟故事演讲

**演讲主题：** `A Person Who Changed Me`

**结构框架：**
```
OPENING (30 秒):
- Hook: 介绍这个人
  "The person I want to talk about is..."
- Context: 你们的关系
  "He/She is my..."

BODY (3 分 30 秒):
- First Meeting: 第一次见面
  "I still remember the first time we met..."
- Key Event: 关键事件（使用五觉细节）
  "That day, I saw/heard/felt..."
- My Change: 我的改变
  "Because of him/her, I..."

CONCLUSION (1 分钟):
- Summary: 总结
  "Looking back, I realize..."
- Gratitude/Insight: 感谢/感悟
  "I want to say thank you..." / "Now I understand..."
```

**准备时间：** 10 分钟
- 学生填写故事框架工作表
- 教师巡回指导

**演讲时间：** 20 分钟
- 每组 4-5 人，组内演讲
- 每人 5 分钟 + 1 分钟反馈

**演讲要求：**
- 必须使用故事弧光结构
- 至少包含 3 个感官细节
- 有明确的情感变化

---

### 80-90 分钟：总结 + 作业布置

**今日要点回顾：**
```
1. 故事弧光：开始→冲突→高潮→结局
2. 五觉细节：视、听、嗅、味、触
3. 情感共鸣：真实 + 细节 = 感动
```

**优秀表现表扬：**
- 列出今天表现突出的学生
- 具体说明亮点

## 💡 教学提示

### 常见问题及应对

**问题 1: 学生故事太平淡**
```
应对：
- 引导找出"转折点"
- 问："那天有什么特别的？"
- 帮助挖掘情感高潮
```

**问题 2: 学生缺乏细节**
```
应对：
- 用五觉法提问
- "你当时看到了什么颜色？"
- "你听到了什么声音？"
- "你心里是什么感觉？"
```

**问题 3: 学生情感表达困难**
```
应对：
- 教师先示范分享个人故事
- 营造安全、信任的氛围
- 允许学生选择舒适的话题
```

### 差异化教学建议

| 学生类型 | 支持策略 |
|----------|----------|
| **英语薄弱** | 提供句型模板，允许使用简单句 |
| **表达内向** | 先小组内分享，再全班展示 |
| **能力强** | 要求更复杂的结构和修辞 |
