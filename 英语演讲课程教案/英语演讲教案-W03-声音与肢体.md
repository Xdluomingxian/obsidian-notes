## ⏰ 教学流程

### 0-15 分钟：作业回顾

**活动：**
1. 展示 2 份优秀 Week2 演讲稿
2. 点评结构运用情况
3. 学生分享写作心得

---

### 15-35 分钟：声音技巧讲解

**Voice Elements 声音四要素：**

| 要素 | 说明 | 练习方法 |
|------|------|----------|
| **Volume 音量** | 让最后一排听清 | 从 1 数到 10，逐渐增大 |
| **Pitch 音调** | 避免单调 | 用不同音调说"Really?" |
| **Pace 语速** | 适中，有变化 | 快慢交替读同一段落 |
| **Pause 停顿** | 强调重点 | 在关键词前后停顿 |

**语调练习：**
```
用不同语调说同一句话：

"I love China."
- 陈述语气（平调）
- 疑问语气（升调）
- 感叹语气（降调）
- 强调语气（起伏）
```

---

### 35-55 分钟：肢体语言训练

**Body Language 肢体五要素：**

| 要素 | 要点 | 错误示例 |
|------|------|----------|
| **Eye Contact 眼神** | 扫视全场，每点 3 秒 | 只看地板/天花板 |
| **Gesture 手势** | 自然，与内容配合 | 双手插兜/抱胸 |
| **Posture 站姿** | 挺直，重心稳定 | 摇晃/倚靠 |
| **Facial Expression 表情** | 与情感一致 | 面无表情 |
| **Movement 移动** | 适度，有目的 | 来回踱步 |

**课堂练习：**
```
1. 眼神练习：
   - 4 人一组，每人说话时与其他人眼神交流
   - 每个点停留 3 秒

2. 手势练习：
   - 列举：数手指
   - 对比：左右手分别代表
   - 强调：握拳/手掌向下

3. 站姿练习：
   - 神奇女侠姿势 2 分钟
   - 感受自信状态
```

---

### 55-80 分钟：古诗英诵练习

**《静夜思》英文译本：**
```
Before my bed, the moonlight glows,
I wonder if it's frost on ground.
I raise my head, the moon I watch,
I lower my head, homesick I'm found.

- Translated by Xu Yuanchong
```

**练习步骤：**
```
1. 听教师示范（1 遍）
2. 全班齐读（2 遍）
3. 分组练习（5 分钟）
   - 注意语调起伏
   - 加入适当手势
   - 控制语速停顿

4. 小组展示（每组 1 代表）
5. 同伴互评
```

**表演要求：**
- 站立姿势端正
- 有眼神交流
- 配合诗句做手势
- 语速适中，有停顿

---

### 80-90 分钟：总结 + 作业布置

**今日要点：**
```
1. 声音四要素：Volume, Pitch, Pace, Pause
2. 肢体五要素：Eye, Gesture, Posture, Face, Move
3. 声音 + 肢体 = 完整表达
```

## 💡 教学提示

### 常见问题及应对

**问题 1: 学生声音太小**
```
应对：
- 让全班一起大声数数
- 玩"传话"游戏
- 鼓励"想象对最后一排说话"
```

**问题 2: 学生肢体僵硬**
```
应对：
- 先做热身运动
- 从简单手势开始
- 教师多示范
```

**问题 3: 学生不敢看别人**
```
应对：
- 先看对方额头
- 找友善的面孔
- 逐渐增加眼神接触时间
```
