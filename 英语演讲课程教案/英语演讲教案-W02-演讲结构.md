> **主题**: 掌握演讲的三段式结构  
> **课时**: 2 课时（90 分钟）  
> **日期**: ______年______月______日

## 🎯 教学目标

### 知识目标
- 理解演讲的开场白、主体、结尾三部分
- 掌握 3 种开场白技巧
- 掌握 2 种结尾升华方法

### 能力目标
- 能够组织 3 分钟微型演讲
- 能够运用结构框架组织内容

### 素养目标
- 培养逻辑思维能力
- 增强表达的结构意识

## ⏰ 教学流程

### 0-15 分钟：作业回顾 + 优秀视频展示

**活动：**
1. 播放 2-3 个优秀 Week1 作业视频
2. 教师点评亮点
3. 学生分享家乡故事的收获

**点评要点：**
```
✅ 内容具体（有细节）
✅ 情感真挚（有故事）
✅ 表达自然（有眼神交流）
```

---

### 15-35 分钟：演讲结构讲解

**三段式结构：**
```
┌─────────────────────────────────────┐
│  Opening (开场白)  10-15%           │
│  - Hook the audience                │
│  - State your topic                 │
├─────────────────────────────────────┤
│  Body (主体)  70-80%                │
│  - Point 1 + Example                │
│  - Point 2 + Example                │
│  - Point 3 + Example                │
├─────────────────────────────────────┤
│  Conclusion (结尾)  10-15%          │
│  - Summarize                        │
│  - Call to action / Final thought   │
└─────────────────────────────────────┘
```

**3 种开场白技巧：**

| 技巧 | 示例 | 适用场景 |
|------|------|----------|
| **提问式** | "Have you ever wondered why...?" | 引发思考 |
| **故事式** | "Last year, I experienced something..." | 情感共鸣 |
| **数据式** | "Did you know that 80% of...?" | 说服论证 |

**2 种结尾升华方法：**

| 方法 | 示例 | 效果 |
|------|------|------|
| **呼应开头** | 回到开场的问题/故事 | 首尾呼应 |
| **号召行动** | "So let's start today..." | 激发行动 |

---

### 35-50 分钟：TED 演讲分析

**观看视频：** 《中国年轻人怎么看世界》（3 分钟片段）

**分析任务：**
```
小组讨论（4 人一组）：

1. 演讲者用什么方式开场？
2. 主体部分有几个要点？
3. 结尾如何升华？
4. 你最喜欢哪一部分？为什么？
```

**分享环节：**
- 每组选代表分享分析结果
- 教师补充点评

---

### 50-80 分钟：3 分钟微型演讲练习

**演讲主题：** `A Chinese Tradition I Love`

**结构框架：**
```
Opening (30 秒):
- Hook: Question/Story/Quote
- Topic: "Today I want to share..."

Body (2 分钟):
- Point 1: What is this tradition?
- Point 2: Why do I love it?
- Point 3: A personal memory

Conclusion (30 秒):
- Summary: "In conclusion..."
- Final thought: Why it matters
```

**准备时间：** 10 分钟
- 学生填写结构工作表
- 教师巡回指导

**演讲时间：** 20 分钟
- 每组 4-5 人，组内演讲
- 每人 3 分钟 + 1 分钟反馈

---

### 80-90 分钟：总结 + 作业布置

**今日要点回顾：**
```
1. 三段式结构：Opening-Body-Conclusion
2. 3 种开场白技巧
3. 2 种结尾升华方法
4. 结构让演讲更清晰
```

## 📝 作业布置

### 必做作业
```
撰写演讲稿：My Favorite Chinese Festival

要求：
1. 使用三段式结构
2. 至少使用 1 种开场白技巧
3. 主体部分有 2-3 个要点
4. 结尾有升华
5. 字数：300-400 词

提交格式：Word 文档
截止时间：下节课前
```

### 选做作业
```
分析 1 个你喜欢的演讲（TED/演讲比赛）
- 记录其结构
- 标注开场和结尾技巧
- 写 100 词分析心得
```
